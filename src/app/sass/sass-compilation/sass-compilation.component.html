Compilation

This deals with how you compile .css files from .scss files using sass command. You'll need to tell Sass which file to build from and where to output to. The basic command is to run sass input.scss output.css from your terminal or command prompt.

C:\>sass scss/style.scss css/style.css /* For globally installed sass */
C:\Users\Bikash\source\repos\user_interface\style>npx sass scss/main.scss css/main.css /* For locally installed sass run this command from the project root. */

Watch For Change

--watch keeps watching for change in the .scss file, and compiles .css and .css.map files immediately upon change detection in the .scss file.

C:\>sass --watch scss/style.scss css/style.css

Output To Directory

You can watch and output to directories by using folder paths as your input and output, and separating them with a colon. Sass would watch all files in the app/sass folder for changes, and compile CSS to the public/stylesheets folder.

C:\>sass --watch app/sass:public/stylesheets
