Map

Maps in Sass hold keys and values. They are written (key: value, key: value). Keys need to be unique. A map with no keys and values is written () (empty parentheses), which is also an empty list.

Sass
CSS
$map, map.get($map, $key)

$font-weights is a map with keys regular, medium, bold and values 400, 500, 700 respectively.

$primary-color: #272727;
$text-color: #fff;
$font-weights: (
  "regular": 400,
  "medium": 500,
  "bold": 700
)

body {{'{'}}
  background: $primary-color;
  color: $text-color;
  font-weight: map.get($font-weights, "bold");
}
body {{'{'}}
  background: #272727;
  color: #fff;
  font-weight: 700;
}
