Inheritance

This deals with inheritance.


% is a placeholder selector and is similar to class selectors. Placeholder selectors don't show up in the generated CSS. @extend is used to inherit placeholder selectors.

Sass
CSS
%icon{{'{'}}
  transition: background-color ease .2s;
  margin: 0 .5em;
}

.error-icon {{'{'}}
  @extend %icon;
  background-color: red;
  color: white;
}

.info-icon {{'{'}}
  @extend %icon;
  background-color: yellow;
  color: black;
}
.error-icon, .info-icon{{'{'}}
  transition: background-color ease .2s;
  margin: 0 .5em;
}

.error-icon {{'{'}}
  background-color: red;
  color: white;
}

.info-icon {{'{'}}
  background-color: yellow;
  color: black;
}
