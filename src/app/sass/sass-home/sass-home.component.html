Sass

This is a tutorial on Sass.


What is Sass?

  • Sass stands for Syntactically Awesome Style Sheets.
  • It is a CSS preprocessor / extension (extends natural capabilities of css) or sometimes it is called a css language.
  • Lets you use features that do not exist in CSS.
  • Sass (.scss) files are compiled to CSS.
  • .scss file is preferred to .sass file. .scss is backward compatible.


File Structure

  1. Create scss folder and and style.scss file.
  2. write the folloing code in style.scss file.
    $font-stack: Arial, Helvetica, sans-serif;
    $light-color: #f4f4f4;

    * {{'{'}}
        box-sizing: border-box;
        margin: 0;
        padding: 0;
    }

    body {{'{'}}
        background-color: $light-color;
        font-family: $font-stack;
        line-height: 1.6;
    }
  3. Compile style.css from style.scss. --watch will keep watching for any change in the scss/style.scss file.
    sass --watch scss/style.scss css/style.css
