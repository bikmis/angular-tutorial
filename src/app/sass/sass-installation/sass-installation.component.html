Installation

Install Sass using the following npm command at the command prompt. To run the npm command, you need to first install Node.js.

npm install -g sass     /*-g for installation as a global package*/
or
npm i -g sass     /*i for install and -g for installation as a global package*/
npm install sass     /*installs sass as a local package in the project. Change directory into the project root and run this command.*/
