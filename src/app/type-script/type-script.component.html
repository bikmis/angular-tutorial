Type Script


| (Pipe Symbol or Vertical Bar)
A pipe can combine several types to denote a Union Type. A variable with a Union Type can be any of the types combined by the pipe.

In the code below, x is either string or number.
var x: string | number

Index Signature
The StringArray object can have any number of properties. Each property will be a number, but will be assigned to a string.
interface StringArray {{'{'}}
  [index: number]: string;
}

let myArray: StringArray = ['Bob', 'Fred'];
let myString: string = myArray[0]; // myString is set Bob

Type Assertions (type cast)
A type assertion is like a type cast in other languages. Type assertions use two forms: "angle-bracket" syntax or as-syntax.

"angle bracket" syntax:
let someValue: any = "this is a string";
let strLength: number = (<string>someValue).length;

as-syntax:
let someValue: any = "this is a string";
let strLength: number = (someValue as string).length;
