*ngFor

*ngFor is a for loop in Angular. It is a structural directive attribute. It changes the DOM's structure by adding, removing, or manipulating elements. A structural directive attribute begins with an asterisk(*).

The following array is coded in the component class.
public programingLanguage = ['c', 'c++', 'c#'];

The following *ngFor is coded in the template.
<ul>
  <li *ngFor="let language of programingLanguage; let i=index; let isOdd=odd; let isEven=even"     class="list-group-item" [class.odd]="isOdd" [class.even]="isEven">
    ({{'{{'}}i+1}}) {{'{{'}}language}}
  </li>
</ul>

The Result:
The following is a list of programing languages.
  • ({{i+1}})    {{language}} ----- isOdd is {{isOdd}} & isEven {{isEven}}. ".odd" class is appplied where isOdd is true & ".even" is applied where isEven is true.

*ngIf

When you click the button, it toggles between visible and hidden by setting isVisible to true or false. *ngIf is set to isVisible.

The following is the function coded inside the component class.
public toggleText(event) {{'{'}}
if (this.isVisible) {{'{'}}
    this.isVisible = false;
    event.target.innerHTML = 'Show Text';
  }
else {{'{'}}
    this.isVisible = true;
    event.target.innerHTML = 'Hide Text';
  }
}

The following is the html code in the template.

<a class="btn btn-sm btn-outline-secondary mr-2" (click)="toggleText($event)">Hide Text</a> <div class="d-inline" *ngIf="isVisible">This text toggles between visible and hidden.</div>

This toggles between visible and hidden.


*ngSwitch

Write the following code in the component.
public programingLanguage = ['c', 'c++', 'c#', 'Java', 'JavaScript', 'Python'];
public selectedLanguage: string = "Select a Language";

Write the following code in the template for the dropdown.
<select [(ngModel)]="selectedLanguage">
  <option value="Select a Language">Select a Language</option>
  <option *ngFor="let language of programingLanguage">
    {{'{{'}}language}}
  </option>
</select>

Write the following code in the template for the ngSwitch.
<div [ngSwitch]="selectedLanguage">
  <a *ngSwitchCase="'c'" target="_blank" href="https://www.gnu.org/software/gnu-c-manual/gnu-c-manual.html">Learn c</a>
  <a *ngSwitchCase="'c#'" target="_blank" href="https://docs.microsoft.com/en-us/dotnet/csharp/programming-guide/">Learn c#</a>
  <a *ngSwitchCase="'JavaScript'" target="_blank" href="https://developer.mozilla.org/en-US/docs/Web/JavaScript">Learn JavaScript</a>
  <span *ngSwitchDefault>
    <div *ngIf="selectedLanguage && !(selectedLanguage === 'Select a Language')">This course is not currently available. </div>
  </span>
</div>

Example:
Learn c Learn c# Learn JavaScript
This course is not currently available.
