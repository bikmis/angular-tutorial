Angular has a tool called In Memory Web API. Once installed this library will intercept all your HTTP calls and return mock data for you. You don't have to hardcode anything or rely on the backend team. The tool makes everything seamless even mimicking delayed responses if you want it too. You can set headers, response types, basically everything you might do with a server.

Install the in-memory-web-api as follows:
npm i angular-in-memory-web-api --save
