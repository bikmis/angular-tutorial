Interpolation allows you to include data into the the text between HTML element tags

In the component class we declare and assign properties as follows:
public chapter: {{'{'}} title: string } = {{'{'}} title: 'Component'};

In the template we bind data as follows:
<span>Title of this chpater is "{{'{{'}}chapter.title}}".</span>
<span>Current price is ${{'{{'}}currentPrice}}.</span>

public selectedMonth: string = null;

public changeMonths(event) {{'{'}}
  if (event.target.options.selectedIndex === 0) {{'{'}}
    this.selectedMonth = null;

  </option>

Two-way Binding with [(ngModel)]

The two-way binding syntax is just syntatic sugar for a property binding and an event binding. The NgModel directive allows you to display a data property and update that property when the user makes changes. You need to import FormsModule in the AppModule to make the [(ngModule)] available in the application.

import {{'{'}} FormsModule } from '@angular/forms';"
  ....
  ....
@NgModule({{'{'}}
    ....
    imports: [
        ....
    FormsModule,
        ....
    ]
)}
export class AppModule{{'{'}} }

Example 1:
We will write the following code in the component.
public firstName: string = "Michael";

We will write the following code in the template.
<input [(ngModel)]="firstName" type="text">
<p *ngIf="firstName">You first name is {{'{{'}}firstName}}.</p>

You first name is {{firstName}}.

Example 2:
We will write the following code in the component.
public selected_month: string = "Select a month";

In the following code, ngModel is set to selected_month in a two way binding in the template. Whenever the selected_month's value changes in the component, it will be reflected in the template and vice versa.
<select [(ngModel)] = "selected_month">
  <option value="Select a month">Select a month</option>
  <option *ngFor="let month of months" value="{{'{{'}}month.toLowerCase()}}">
    {{'{{'}}month}}
  </option>
</select>
<p *ngIf="selected_month && selected_month != 'Select a month'">You selected {{'{{'}}selected_month}}.</p>


You selected {{selected_month}}.
