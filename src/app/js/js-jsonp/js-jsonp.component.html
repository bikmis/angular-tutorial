jsonp

The script with the src set to a url and with a callback function myFunc will get the data wrapped in the function myFunc. This is json data padded with a JavaScript function. This is jsonp (json with padding). Cross-domain policy does not allow requesting data from another domain but requesting script from another domain does not have this problem. JSONP uses this and requests data using the script tag instead of the XMLHttpRequest object.
<script>
  function myFunc(data){{'{'}}
    console.log(data);
  }
</script>

<script src="https://jsonplaceholder.typicode.com/users?callback=myFunc"></script>
