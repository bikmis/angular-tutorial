Location of JS


This page discusses where to place JavaScript code.

  • Type in functions in the <script></script> tag inside the body tag at the bottom and open the page in a browser.
    <script>
        console.log("something");
        alert("something");
    </script>;
  • Type in the following code in the address bar of the browser and hit return.
    javascript:alert("Hello World");console.log("Hey");
  • Open the Console in a browser by right-clicking and clicking Inspect. Inside Console type in the following and hit return. It prints Hello World and then undefined. console.log and alert() return void and so undefined is printed. If you use prompt() and hit return and type in something in the input, that will be printed and undefined is not printed as prompt() return a string, not void.
    console.log("something");alert("Hello World");
  • Create a .js file and write javascript code in it and call it with the code below written before the closing body tag.
    <script src="./js/youtube.js"></script>
  • You can also put in the <head></head> section a script tag with javascript code and/or with a src attribute linking to an external javascript file. But you may not be able to reference html elements from your script in the <head></head> section, so it is a best practice to put the javascript code before the closing body tag as the html elements in the body will already be rendered. If you put the script in the <head></head> section, use one of the following to have access to the DOM elements:
    window.addEventListener("DOMContentLoaded", function() {{'{'}} ... });
    window.addEventListener("load", function() {{'{'}} ... });
    <script defer src="./js/youtube.js"></script>
  • Outside browsers, we can run JavaScript in NodeJs, which is built on Chrome's V8 JavaScript engine.
