window


This page discusses the window object. window interface contains the document property loaded in the window or tab. window is a global variable. Even in a tabbed browser, some properties and methods still apply to the overall window that contains the tab, such as resizeTo() and innerHeight. Generally, anything that can't reasonably pertain to a tab pertains to the window instead.

console

console is an object on window. console.log("argument") function prints in the Console.
var consoleLogId = document.getElementById("console-log");
consoleLogId.addEventListener("click", function(){{'{'}}
  console.log("You clicked on an element with the id 'console-log'.")
});
console.log(param1, param2, ...) can print multiple parameters in the console. The following code prints 2 3;
var x = 2, y = 3;
console.log(x, y);

confirm()

window.confirm(message) pops up a message with ok and cancel buttons. If ok is pressed, true is returned. For cancel false is returned.

google.addEventListener("click", function(){{'{'}}

});

document

window.document or document returns a reference to the document that the window contains.

history

history.back();
history.go(1); history.forward();

location

Gets/sets the location, or current URL, of the window object.

localStorage

Returns a reference to the local storage object used to store data that may only be accessed by the origin that created it.

name

Gets/sets the name of the window.

opener

Returns a reference to the window that opened this current window.

parent

Returns a reference to the parent of the current window or subframe.

status

Gets/sets the text in the statusbar at the bottom of the browser.

close()

Closes the current window.

open()

Opens a new window.

postMessage()

Provides a secure means for one window to send a string of data to another window, which need not be within the same domain as the first.

print()

Opens the Print Dialog to print the current document.

resizeTo(), resizeBy()

Dynamically resizes window.
Resizes the current window by a certain amount.

scroll(), scrollBy(), scrollTo()

Scrolls the window to a particular place in the document.
Scrolls the document in the window by the given amount.
Scrolls to a particular set of coordinates in the document.

atob()

Decodes a string of data which has been encoded using base-64 encoding.

btoa()

Creates a base-64 encoded ASCII string from a string of binary data.

addEventListener()

Register an event handler to a specific event type on the window.

clearInterval()

Cancels the repeated execution set using setInterval().

clearTimeout()

Cancels the delayed execution set using setTimeout().

setInterval

Schedules a function to execute every time a given number of milliseconds elapses.

setTimeout

Schedules a function to execute in a given amount of time.

onaffterprint

Called when the print dialog box is closed.

onbeforeprint

Called when the print dialog box is opened.

onblur

Called after the window loses focus, such as due to a popup.

onclick

Called after the ANY mouse button is pressed & released

ondblclick

Called when a double click is made with ANY mouse button.

onclose

Called after the window is closed

onerror

Called when a resource fails to load OR when an error occurs at runtime.

onfocus

Called after the window receives or regains focus.

onhashchange

An event handler property for hashchange events on the window; called when the part of the URL after the hash mark ("#") changes.

oninput

Called when the value of an input element changes

onkeydown

Called when you begin pressing ANY key.

onkeypress

Called when a key (except Shift, Fn, and CapsLock) is in pressed position.

onkeyup

Called when you finish releasing ANY key.

onload

Called after all resources and the DOM are fully loaded. WILL NOT get called when the page is loaded from cache, such as with back button.

onmousedown

Called when ANY mouse button is pressed.

onmousemove

Called continously when the mouse is moved inside the window.

onmouseout

Called when the pointer leaves the window.

onmouseover

Called when the pointer enters the window

onmouseup

Called when ANY mouse button is released

onoffline

Called when network connection is lost.

ononline

Called when network connection is established.

onpagehide

Called when the user navigates away from the page, before the onunload event.

onpageshow

Called after all resources and the DOM are fully loaded.

onpopstate

Called when a back button is pressed.

onreset

Called when a form is reset

onresize

Called continuously as you are resizing the window

onscroll

Called when the scroll bar is moved via ANY means. If the resource fully fits in the window, then this event cannot be invoked

onwheel

Called when the mouse wheel is rotated around any axis

onselect

Called after text in an input field is selected

onstorage

Called when there is a change in session storage or local storage.

onsubmit

Called when a form is submitted

onunload

Called when the user navigates away from the page.

error

Fired when a resource failed to load, or can't be used. For example, if a script has an execution error or an image can't be found or is invalid.

languagechange

Fired at the global scope object when the user's preferred language changes.

orientationchange

Fired when the orientation of the device has changed.

resize

Fired when the window has been resized.

storage

Fired when a storage area (localStorage or sessionStorage) has been modified in the context of another document.

animationend

Fired when an animation has completed normally.

animationstart

Fired when an animation starts.

animationiteration

Fired when an animation iteration has completed.

copy, cut, paste

Fired when the user initiates a copy, cut or paste action through the browser's user interface.

online, offline

online: Fired when the browser has gained access to the network and the value of navigator.onLine has switched to true.

offline: Fired when the browser has lost access to the network and the value of navigator.onLine has switched to false.

blur

Fired when an element has lost focus.

focus

Fired when an element has gained focus.

hashchange

Fired when the fragment identifier of the URL has changed (the part of the URL beginning with and following the # symbol).

pagehide

Sent when the browser hides the current document while in the process of switching to displaying in its place a different document from the session's history. This happens, for example, when the user clicks the Back button or when they click the Forward button to move ahead in session history.

pageshow

Sent when the browser makes the document visible due to navigation tasks, including not only when the page is first loaded, but also situations such as the user navigating back to the page after having navigated to another within the same tab.

popstate

Fired when the active history entry changes.

beforeunload

Fired when the window, the document and its resources are about to be unloaded.

DOMContentLoaded

Fired when the document has been completely loaded and parsed, without waiting for stylesheets, images, and subframes to finish loading.

load

Fired when the whole page has loaded, including all dependent resources such as stylesheets, images.

unload

Fired when the document or a child resource is being unloaded.

message

Fired when the window receives a message, for example from a call to Window.postMessage() from another browsing context.

messageerror

Fired when a Window object receives a message that can't be deserialized.

afterprint

Fired after the associated document has started printing or the print preview has been closed.

beforeprint

Fired when the associated document is about to be printed or previewed for printing.

transitionend

Fired when a CSS transition has completed.

transitionrun

Fired when a CSS transition is first created.

transitionstart

Fired when a CSS transition has actually started.
