Data Type


This page discusses data types.

Primitive

A primitive value or primitive data type is data that is not an object and has no methods. There are 6 primitive data types:
  1. string
  2. number
  3. bigint
  4. boolean
  5. undefined
  6. symbol
null looks like primitive, but is a special case for every object. Primitives are immutable, i.e they cannot be changed. Don't confuse primitive itself with a variable assigned a primitive value. The variable maybe reassigned a new value, but the existing value can not be changed in the ways that objects, arrays, and functions can be altered. A primitive (value) can be replaced but not directly altered.

In JavaScript, data are dynamically-typed.

let x = 3; typeof(x); //number
x = "Jack"; typeof(x); //string
x = true; typeof(x); //boolean
x = 12n; typeof(x); //bigint. bigints are integers with n.
var y; typeof(y); //undefined
let z = new Date(); typeof(z); //object

Object

An object is a collection of properties, and a property is an association between a name (or key) and a value. A property's value can be a function, in which the property is know as a method. objectName.propertyName gives the value of the property.

  • let myCar = new Object();
    myCar.make = "Honda";
    console.log(myCar.make); //Honda

  • let myCar = {{'{'}} make: "Honda"};
    console.log(myCar.make); //Honda
    myCar.color = "Red";
    console.log(myCar.color); //red
    console.log(myCar["color"]); //red
    myCar["price"] = "$30,000";
    console.log(myCar.price); //$30,000

  • let student = {{'{'}}
      name: "Jack",
      age: 22,
      isFunny: function(){{'{'}}
        if(this.age > 20) {{'{'}}
          return true;
        } else {{'{'}}
          return false;
        }
      }
    }

    let isFunny = student.isFunny();
    console.log(isFunny);

  • let now = new Date();
    console.log(now, typeof(now)); //typeof(now) is object

  • let grades = [10, 30, 40, 60];
    console.log(grades[0], typeof(grades)); //typeof(grades) is object

this (Object Reference)

"this" can be used within a method to refer to the current object. A method is a variable in an object that is set to a function. In other words, a method is a function which is a property of an object.

}

  name: "Michael",
