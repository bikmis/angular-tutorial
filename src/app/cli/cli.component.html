CLI

CLI stands for command line interface. Angular CLI has the following commands:
  • npm install -g @angular/cli --> installs angular cli and becomes available globally in the OS. If angular cli is already installed and you run this command again, it will intall the latest package overwriting the existing installation.
  • ng new my-new-app --> this will create a new workspace and project under the folder my-new-app.
  • ng new --> this will create a new project and will prompt you with questions including project name.
  • ng serve --> this will test your app locally at http://localhost:4200/ while developing. It builds and serves your app, rebuilding on file changes.
  • ng generate ... this will generate components, routes, services and pipes with test shells.
  • ng build --> builds the app and compiles it into an output directory named dist/.
  • ng test --> runs unit tests in a project.
  • ng update --> This command will analyse package.json and update the packages from dependencies.
  • ng update @angular/core --> You run this command from the workspace root directory to update the angular core to the latest stable version.
  • ng update rxjs --> updates rxjs.
  • ng version or ng v --> outputs angular cli version.
  • ng config --> Gets or sets angular configuration values in the angular.json file for the workspace.
  • ng analytics --> configures the gathering of angular cli usage metrics.
  • ng add --> adds support for an external library to your project.
