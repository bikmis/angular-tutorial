Angular Material

  • ng add @angular/material This will install the Angular Material, the Component Dev Kit (CDK), Angular Animations. It will also ask you for the theme and browser animations. It will add project dependencies to package.json, Roboto font to index.html, add a few global CSS styles to
    • Remove margins from body
    • Set height: 100% on html and body
    • Set Roboto as the default application font
