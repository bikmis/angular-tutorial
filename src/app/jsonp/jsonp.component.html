Apps can use the HttpClient to make JSONP requests across domains when a server doesn't support CORS protocol. In Angular, use JSONP by including HttpClientJsonpModule in the NgModule imports. JSONP means json data padded with a callback function. jsonp call creates a script tag with a source URL and defines a callback to be triggered when the requested script with data is fetched from the server.
import {{'{'}} HttpClientModule, HttpClientJsonpModule } from '@angular/common/http';
