Styling

If you want to use css:

  • At the time of project creation, choose css for styling.
  • Create a css file in the "src/styles/site" folder and import it in the src/styles.css with
    @import "./styles/site/homePage.css";

If you want to use scss:

  • At the time of project creation, choose scss for styling.
  • Create scss partial files such as _variable.scss in the "src/styles/site" folder and import them in the styles.scss with
    @import "./style/site/variable";

If you want to use bootstrap (scss):

  • At the time of project creation, choose scss for styling.
  • Keep bootstrap related files and folders in the "src/styles/bootstrap" folder and import them in the styles.scss with
    @import "./styles/bootstrap/bootstrap";
  • Create site-related scss partial files such as _variable.scss etc and keep them in the "src/styles/site" folder and import them in the styles.scss with
    @import "./styles/site/variable";
    Import of _variable.scss file should be before the import of bootstrap.scss.
  • "ng serve" / "ng build" will build css files from styles.scss automatically.

If you want to create new projects with scss in the future, you can change the config as follows:
ng config --global defaults.styleExt=scss
