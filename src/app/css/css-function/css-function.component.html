Functions

This deals with css functions.



calc() Function

calc() function performs a calculation to be used as the property value. Operators like + - * / can be used.

#div1 {{'{'}}
  --variable-width: 30px;   width: calc(50px + 100px);
  or
  width: calc(100% - 100px);
  or
  width: calc(3rem * 2);
  or
  width: calc(var(--variable-width) + 10px);
}


var() Function

The var() css function is used to insert the value of a custom property (sometimes called a css variable)

:root {{'{'}}
  --color-red: red;
  --color-blue: blue:
}

h1 {{'{'}}
  color: var(--color-red);
}

p {{'{'}}
  color: var(--color-blue);
}
