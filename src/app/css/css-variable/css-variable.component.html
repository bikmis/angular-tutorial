Variables / Custom Properties

This deals with custom properties/variables in css.



Global Variable

:root is a pseudo-class selector used to select the root element of the page/document. Normally HTML is the root of the document, but :root has higher specificity. You can create a global custom properties / variables by putting them in the :root class.

:root {{'{'}}
  /*two dashes let the browser know this is a custom property / variable.*/
  --primary-clr: #456; /*This is a global variable/property that can be used anywhere in this document.*/
  --spacer: 1rem;
  --spacer-md: calc(var(--spacer) * 2);
  --spacer-lg: calc(var(--spacer) * 3);
}

body {{'{'}}
  color: var(--primary-clr); /*var() is a function that inserts value of custom property.*/
}

.top-section {{'{'}}
  --secondary-clr: steelblue; /*This variable/property can be used only inside of .intro class.*/
  color: var(--secondary-clr, yellow); /*if --secondary-clr does not work, use yellow.
  This is useful for debugging issues.*/
}

.middle-section {{'{'}}
  --primary-clr: blue; /*global variable --primary-clr has be redefined here.*/
  color: var(--primary-clr);
}
